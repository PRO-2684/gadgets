## 🪄 功能与配置

> 用 `*` 标记的配置项需要刷新以生效

- **🎨 外观**
    - **⬅️ 侧栏管理**: 允许你隐藏侧栏中的各个部分，包括：自定义板块、外部链接、类别、标签、消息、频道、直接消息、聊天、底部菜单
- **♿ 辅助功能**
    - **👆 增大点击区域**: 增大帖子列表中各帖子的可点击区域 (仅支持左键)
    - **🔢 显示楼层**: 在回复右侧显示楼层信息
    - **👤 @用户名**: 在用户名前添加 @ 符号
