# Rettiwt Key Export

Export API_KEY for use with [Rettiwt-API](https://github.com/Rishikant181/Rettiwt-API). UserScript version of the [Rettiwt-Auth-Helper extension](https://github.com/Rishikant181/Rettiwt-Auth-Helper).

## 📦 Installation

- [GitHub](https://github.com/PRO-2684/gadgets/raw/main/rettiwt-key-export/rettiwt-key-export.user.js)

## 🪄 Usage

- Right-click on any `x.com` page. You need to be logged in.
- Select `Tampermonkey` - `Export Cookies`.
- `API_KEY` will be copied to your clipboard.
