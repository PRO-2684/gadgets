# Disco

Hides annoying items on Discord.

## Features

- Hide Super Reactions: Hides the custom animation of super reactions.
- Hide Nitro AD: Hides various ADs/buttons for Nitro. "Settings - Billing - Nitro" entry is left intact.
- Hide Server Boosts: Hides most of boosts interface.
- Hide Top Bar: Hides the top-most title bar.
