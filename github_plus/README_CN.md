# GitHub Plus

## 🪄 功能与配置

- **Debug**: 启用调试模式。
- **Release Count**: 要查询的最大 Release 个数（从最新开始）。
  - 数字越高，加载时间可能越长。
  - 数字越低，如果您继续导航到下一页，您将越早看到一个不带有我们的附加信息的 Release。
- **Release Downloads**: 显示 Release 中文件的下载次数。
- **Release Uploader**: 显示上传 Release 中文件的用户。
